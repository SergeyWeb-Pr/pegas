Двухстоечные ленточные пилы для перпендикулярных пропилов. Прочная машина, предназначенная для резки твердых материалов с акцентом на производительность резки. Гидравлический агрегат управляет всеми функциями пилы. Мощный привод с частотным преобразователем для непрерывной регулировки окружной скорости ленты.

ЧПУ= комплектный численно-управляемый станок с подачей материала с возможностью ввода нескольких программ установки количества и длины нарезанных заготовок.
